### `ASL_calculateDiffMap`

Calculate difference map and write out the .img/.hdr files

!!! info "Inputs"
    - path_temp, asl_paras

!!! success "Outputs"
    - difference image and header files

!!! example "Syntax"
    ```matlab
    ASL_calculateDiffMap(path_temp,asl_paras)
    ```
